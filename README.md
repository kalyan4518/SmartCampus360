🏫 Smart Campus Ecosystem
Smart Campus Ecosystem is a full-stack web application designed to simplify campus activities and improve communication between students, faculty, and administrators.

🚀 Features

🔐 Authentication
Three user roles: Student, Faculty, and Admin.

Student Login – Accessible only using KLH domain email IDs.  
Faculty Login – Accessible using any valid email ID.  
Admin Login – Provides full access to manage campus data.

👩‍🎓 Student Portal
- View upcoming events and announcements.
- Access resources uploaded by faculty (downloadable).
- Explore and join clubs created by the admin.
- Use the Lost & Found section to report or recover items.
- View faculty responses to submitted feedback.

👨‍🏫 Faculty Portal
- Respond to student feedback (visible to students after submission).
- Upload study resources for students.
- View announcements and events.

🛠️ Admin Panel
- Create and manage announcements, clubs, and events.
- Track event participation.
- Update or remove clubs and event data.

🧰 Tech Stack
Frontend: React.js  
Backend: Node.js / Express.js  
Database: MongoDB  
Authentication: currently Dummy
📸 Landing Page
Users start from the landing page where they can log in as Student, Faculty, or Admin.

💡 Summary
A unified digital ecosystem for the campus community that connects students, faculty, and administrators through real-time updates, collaboration tools, and resource sharing.
